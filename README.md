## ğŸš€ Getting Started

### Prerequisites
- A modern web browser
- (Optional) Local web server for development

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SaadAbdElGhaffar/interactive-landing-page.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd interactive-landing-page
   ```

3. **Open the project**
   - Simply open `index.html` in your web browser
   - Or use a local server for development:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (if you have live-server installed)
     live-server
     ```

4. **View the website**
   - Open your browser and go to `http://localhost:8000` (if using a server)
   - Or directly open the `index.html` file

## ğŸ¨ Customization

### Colors
The project uses CSS custom properties for easy theming:
```css
:root {
    --main-color: #2196f3;
    --main-color-alt: #1787e0;
    --main-transition: 0.3s;
    --section-color: #ececec;
}
```

### Content
- Update the HTML content in `index.html`
- Replace images in the `/image` folder
- Modify styles in `css/main.css`
- Adjust JavaScript functionality in `js/main.js`

### Responsive Breakpoints
```css
/* Tablet */
@media (min-width: 768px) { /* styles */ }

/* Desktop */
@media (min-width: 992px) { /* styles */ }

/* Large Desktop */
@media (min-width: 1200px) { /* styles */ }
```
